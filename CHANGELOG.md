# История изменений

## v0.1 — первый играбельный релиз

Полный порт с JavaScript на C++ (raylib). Что внутри:

- **Движок:** окно, игровой цикл, камера 2D, deltaTime
- **Мир:** тайловая карта, коллизии, процедурная генерация (комнаты + коридоры)
- **Игрок:** движение WASD/геймпад, HP, i-frames, антибаг застревания в стенах
- **Враги:** типы (обычные / быстрые / танки), боссы с рывками, ИИ с A*
- **Бой:** автоатака, снаряды, эволюция оружия (веер + пробитие)
- **Прокачка:** опыт, уровни, выбор улучшений
- **Контент:** лут (лечение / усиление), ловушки
- **Интерфейс:** HUD (полоски HP/опыта, таймер), меню, пауза, Game Over, кириллический шрифт
- **Звук:** музыка меню/карты, SFX, голоса боссов с субтитрами
- **Сохранения:** рекорд (время/уровень) и настройка громкости
- **Оптимизация:** пулы объектов для врагов, снарядов, опыта и лута
