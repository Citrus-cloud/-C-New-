# Dungeon Survivors: D20 — C++ edition

Порт игры **Dungeon Survivors: D20** с JavaScript на **C++** на движке [raylib](https://www.raylib.com/).
Стиль — *bullet heaven* / survivors-like (как **Vampire Survivors**): большая карта, толпы врагов, автоатака, опыт и прокачка.

> **Версия: v0.2 (в разработке)** — добавляем настоящую графику (спрайты, тайлсет, большой мир и эффекты) поверх играбельного ядра v0.1.

## ✨ Возможности
- Процедурная генерация подземелий (комнаты + коридоры, новая карта каждый забег)
- Разные типы врагов: обычные, быстрые, танки и **боссы с рывками**
- ИИ врагов с поиском пути **A*** и проверкой линии видимости
- Автоатака, **эволюция оружия** (веер из снарядов + пробитие)
- Опыт, уровни и выбор улучшений
- Лут (лечение / усиление оружия) и ловушки
- HUD с полосками HP/опыта и таймером
- Главное меню (лагерь), пауза и экран Game Over
- Звук, музыка и **голоса боссов с субтитрами**
- Сохранение рекорда и настройки громкости
- Пулы объектов (*object pooling*) для производительности

## Графика v0.2 (в работе)
- **Спрайты и анимации**: менеджер текстур с кэшем и универсальная система анимации по спрайт-листам
- **Игрок**: стойка / ходьба / получение урона, отражение по направлению
- **Враги и боссы**: спрайты по типам, анимация ходьбы и смерти, телеграф рывка босса, полоски здоровья
- **Мир**: тайлсет (пол/стены/декор), отрисовка тайлами с culling, вариативность тайлов, увеличенная карта (96×72) и детерминированный фундамент под чанковую (бесконечную) генерацию

Вся графика необязательна: если PNG-файлов нет, игра рисует цветные квадраты (как в v0.1) и остаётся полностью играбельной.

## 🎮 Управление
- **WASD** / стрелки — движение (атака автоматическая)
- **1 / 2 / 3** — выбор улучшения при повышении уровня
- **ESC** — пауза / продолжить
- **ENTER** — начать игру / вернуться в меню
- **← / →** в меню — громкость
- Поддерживается геймпад

## 🔨 Сборка
Нужны **CMake** (3.15+) и компилятор C++17. raylib подтягивается автоматически через FetchContent.

```bash
cmake -B build
cmake --build build
```

### Релизная сборка под Windows (.exe)
```bash
cmake -B build -DCMAKE_BUILD_TYPE=Release
cmake --build build --config Release
```
Папка `assets` копируется рядом с экзешником автоматически. Для раздачи копируй `.exe` вместе с папкой `assets`.

## 🎨 Ассеты (спрайты, тайлы, звук, шрифт)
Всё необязательно — без файла игра использует fallback.
- **Спрайты** (игрок, враги, боссы): `assets/sprites/` — см. `assets/sprites/README.md`
- **Тайлы** (пол/стены/декор): `assets/tiles/` — см. `assets/tiles/README.md`
- **Шрифт** с кириллицей: `assets/fonts/game.ttf` (иначе русский текст = квадратики)
- **Музыка/звуки/голоса**: `assets/audio/` — см. `assets/README.md`

## 📁 Структура проекта
```
src/
  main.cpp          — игровой цикл и состояния (меню/игра/пауза/Game Over)
  textures.*        — менеджер текстур (загрузка и кэш PNG)
  animation.*       — универсальная анимация по спрайт-листам
  player.*          — игрок (движение, HP, i-frames, спрайты)
  tilemap.*         — тайловая карта, тайлсет, culling, процедурная генерация
  enemy.*           — враги и типы (спрайты, анимации, полоски HP, босс с рывком)
  spawner.*         — волны врагов и боссы, пул объектов
  projectile.*      — снаряды (с пробитием)
  combat.*          — оружие, автоатака, эволюция, выпадение лута
  pickups.*         — сферы опыта
  loot.*            — лут (лечение / усиление)
  upgrades.*        — улучшения при левел-апе
  pathfinding.*     — поиск пути A* и линия видимости
  traps.*           — ловушки
  audio.*           — звук, музыка, голоса боссов
  hud.*             — интерфейс, меню, экраны
  savegame.*        — сохранение рекорда и настроек
assets/             — спрайты, тайлы, шрифты и аудио
CMakeLists.txt      — сборка (raylib через FetchContent)
```
