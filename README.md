# Dungeon Survivors: D20 — C++ edition

Буллет-хевен (в духе Vampire Survivors) на **C++ и raylib**. Это порт моей JavaScript-игры *Dungeon Survivors: D20* на нативный движок — с перспективой на масштабный проект.

> **Текущая версия: v0.3** — углубление механик, способности, поведение врагов и отладочные инструменты (см. CHANGELOG).
>
> **Предыдущая: v0.2** — графика, большой мир и контент.

## Играть

- Движение — **WASD** или стрелки.
- Атака автоматическая — оружие само бьёт по ближайшим врагам.
- Уворот/рывок — **Пробел** или **Shift**: короткий рывок с кадрами неуязвимости и кулдауном.
- **ESC** — пауза.
- На уровне выбирай улучшение клавишами **1 / 2 / 3**.

### Экраны лагеря (главное меню)

- **ENTER** — начать вылазку.
- **C** — выбор героя и карты.
- **U** — мета-прокачка между забегами (за монеты).
- **O** — настройки (громкость музыки/звука раздельно, полный экран).

### Отладка и инструменты (v0.3)

- **F1** — справочный оверлей: сводка забега (время, уровень, HP, оружие, монеты) и управление.
- **F3** — оверлей с параметрами волн и статусом приёмов (ЗАКРЫТ / ГОТОВ / КД).
- **F4** — вручную создать предупреждающую зону (тест телеграфов).
- **F5** — залп снарядов по игроку (тест дальних атак).
- **F6** — заспавнить врагов со всеми приёмами мобильности (тест Фазы 4).
- **F7** — заспавнить врагов со всеми особыми способностями (тест Фазы 5).
- **F8** — панель живой настройки баланса: меняй урон, темп огня, число снарядов, пробой, скорость и HP прямо в бою (`[` / `]` — изменить, PgUp/PgDn — строка).

## Боссы

Появляются каждые 30 секунд, чередуются:

- **Королева пауков** — призывает миньонов.
- **Чёрный рыцарь** — рывки к игроку.

Каждый босс появляется с репликой и голосовой озвучкой (если добавлены аудиофайлы).

## Поведение врагов (v0.3)

Обычные враги при появлении могут получить один из приёмов перемещения (шанс и доступность по времени забега настраиваются в `tuning.h`):

- **Прыжок-наскок** — враг прыгает к игроку по дуге; приземление создаёт зону удара по площади.
- **Телепорт** — издалека враг исчезает и появляется рядом с игроком.
- **Блинк** — серия коротких телепортов-рывков в сторону игрока.
- **Рывок** — враг замирает, показывая линию-предупреждение пути, затем стремительно бросается вперёд.
- **Фланг** — держит дистанцию и заходит сбоку, а не лезет в лоб.

Чёрный рыцарь (босс) использует тот же рывок с предупреждающей линией пути.

### Особые способности врагов (v0.3)

Независимо от приёма перемещения враг может получить одну из особых способностей (шанс и веса — в `tuning.h`):

- **Щит/неуязвимость** — периодически включает щит и на несколько секунд блокирует весь урон.
- **Разделение при смерти** — погибая, враг распадается на несколько мелких быстрых осколков.
- **Аура замедления** — пока игрок рядом, его скорость снижена.
- **Ядовитый след** — враг оставляет лужи, наносящие урон со временем.
- **Лекарь** — периодически лечит союзных врагов в радиусе.

## Бой и реакции (v0.3)

- **Уворот игрока** — рывок в сторону движения (или по направлению взгляда) с короткой неуязвимостью (i-frames) и кулдауном. Скорость, длительность, окно неуязвимости и кулдаун — в `tuning.h`.
- **Отбрасывание врагов** — попадания отталкивают врагов от игрока. Танки сопротивляются отбрасыванию, боссы неуязвимы к нему. Сила и затухание — в `tuning.h`.
- **Статусы и стихийные реакции** — на врагов вешаются статусы, а их сочетания дают реакции: «Перелом» (усиленный урон) и «Пар» (импульсный выброс по площади). Множители и урон — в `tuning.h`.

## Способности

Помимо основного оружия при повышении уровня можно открывать и усиливать:

- **Орбитальные клинки** — вращаются вокруг игрока.
- **Нова** — периодический взрыв по площади.

## Мета-прокачка

За каждый забег начисляются монеты (за уровень и время выживания). В лагере их можно потратить на постоянные бонусы: +здоровье, +урон, +скорость. Прогресс сохраняется.

## Настройка баланса (v0.3)

Все числа боевой системы собраны в одном файле `src/tuning.h`: интервалы и размер волн, времена разблокировки и кулдауны приёмов, шансы типов врагов, параметры телеграфов, мобильности и особых способностей врагов. Меняешь баланс — правишь только его, логика не трогается. Часть параметров можно крутить прямо в бою через панель **F8**.

## Сборка

Нужен CMake 3.15+ и компилятор с поддержкой C++17. raylib 5.5 подтягивается автоматически через FetchContent.

```bash
cmake -B build
cmake --build build
```

Релизная сборка:

```bash
cmake -B build -DCMAKE_BUILD_TYPE=Release
cmake --build build --config Release
```

Ассеты копируются рядом с исполняемым файлом автоматически после сборки.

## Ассеты

Игра работает и без ассетов (вместо спрайтов рисуются фигуры-заглушки). Чтобы увидеть графику и кириллицу, положите файлы в `assets/` (см. README внутри подпапок):

- `assets/sprites/` — спрайты игрока, врагов и боссов (кадры 64x64).
- `assets/tiles/` — пол, стены, декор.
- `assets/vfx/` — взрыв и магический круг.
- `assets/audio/` — музыка, звуки, голоса боссов.
- `assets/fonts/game.ttf` — шрифт с кириллицей (без него русский текст в игре отображается прямоугольниками).

```

  main.cpp           — игровой цикл, состояния, экраны
  player.*           — игрок
  tilemap.*          — карта и генерация

  enemy.* spawner.*  — враги и их порождение
  bosses.h           — таблица боссов
  characters.h       — классы героев
  director.*         — менеджер волн и доступности приёмов (v0.3)
  telegraph.*        — предупреждающие зоны атак / телеграфы (v0.3)
  ranged.*           — дальние атаки и снаряды врагов (v0.3)
  hazards.*          — опасные зоны / ядовитые лужи (v0.3)
  tuning.h           — единый конфиг боя, все числа баланса (v0.3)
  projectile.* combat.* — снаряды и бой
  abilities.*        — способности (орбита, нова)
  upgrades.*         — улучшения при уровне
  pickups.* loot.*   — опыт и лут
  traps.*            — ловушки
  pathfinding.*      — Поиск пути A*
  effects.*          — частицы, тряска, вспышки, цифры урона, виньетка
  audio.*            — звук и музыка
  hud.*              — интерфейс
  savegame.*         — сохранения
  textures.* animation.* — загрузка текстур и анимация
```
