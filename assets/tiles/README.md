# Тайлы (тайлсет карты)

Сюда кладём текстуры пола, стен и декора (PNG, 64×64). Без этих файлов карта рисуется цветными квадратами (как в v0.1).

## Обязательные
| Файл | Назначение |
|------|------------|
| `floor.png` | тайл пола (бесшовный / tileable) |
| `wall.png` | тайл стены |

## Декор (необязательно)
| Файл | Назначение |
|------|------------|
| `decor_bones.png` | кости на полу |
| `decor_puddle.png` | лужа |
| `decor_grass.png` | трава / мох |

## Правила
- Размер 64×64, PNG. Декор — с прозрачным фоном.
- `floor.png` и `wall.png` делай **бесшовными**, чтобы повторялись без видимых границ.
- Для разнообразия код сам отражает тайлы по горизонтали/вертикали детерминированно — одной текстуры хватит.
- Единая палитра с остальным пиксель-артом.
