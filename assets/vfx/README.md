# VFX (эффекты-спрайты)

Сюда кладём спрайт-листы эффектов (PNG, прозрачный фон, кадры в один ряд).
Без этих файлов эффекты рисуются процедурно (расширяющиеся кольца + частицы), игра не падает.

| Файл | Кадры | Назначение |
|------|-------|------------|
| `explosion.png` | 6 | взрыв при смерти врага/босса |
| `magic_circle.png` | 6 | магический круг при левел-апе |

## Правила
- Кадры в один горизонтальный ряд, одинакового размера (например 96×96 на кадр).
- Анимация проигрывается один раз (не зациклена).
- Код сам масштабирует эффект под нужный размер.
