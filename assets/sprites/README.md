# Спрайты (пиксель-арт)

Сюда кладём PNG-спрайты, сгенерированные в ComfyUI. Код сам подхватывает файлы, если они есть; без них всё рисуется квадратами (как в v0.1).

## Общие правила
- Формат: **PNG с прозрачным фоном**.
- Базовый размер кадра: **64×64** px (босс может быть крупнее, напр. 128×128 — код масштабирует автоматически).
- Вид сверху / 3-4 (как камера в игре).
- Единая палитра и один пиксель-арт чекпойнт/LoRA — для консистентности.
- Анимация = **спрайт-лист**: кадры идут в один горизонтальный ряд.
- Рисуй персонажей смотрящими **вправо** — отражение влево код делает сам.

## Игрок (Фаза 1)
| Файл | Кадров в ряду | Назначение |
|------|----------------|------------|
| `player_idle.png` | 4 | стойка на месте |
| `player_walk.png` | 6 | ходьба |
| `player_hurt.png` | 2 | получение урона |

## Враги и боссы (Фаза 2)
| Файл | Кадров в ряду | Назначение |
|------|----------------|------------|
| `enemy_grunt.png` | 4 | обычный враг (ходьба) |
| `enemy_fast.png` | 4 | быстрый враг (ходьба) |
| `enemy_tank.png` | 4 | танк (ходьба) |
| `boss.png` | 6 | босс (ходьба) |
| `enemy_death.png` | 5 | анимация смерти обычных врагов (не зациклена) |
| `boss_death.png` | 6 | анимация смерти босса (не зациклена) |

Пример размера листа: 6 кадров по 64×64 → ширина **384**, высота **64**.

Полоски здоровья рисуются кодом автоматически над врагами и боссами.
