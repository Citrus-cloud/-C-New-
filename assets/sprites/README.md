# Спрайты (пиксель-арт)

Сюда кладём PNG-спрайты, сгенерированные в ComfyUI.

## Общие правила
- Формат: **PNG с прозрачным фоном**.
- Базовый размер кадра: **64×64** px (боссы можно крупнее, напр. 128×128).
- Вид сверху / 3-4 (как камера в игре).
- Единая палитра и один пиксель-арт чекпойнт/LoRA — для консистентности.
- Анимация = **спрайт-лист**: кадры идут в один горизонтальный ряд.

## Игрок (Фаза 1)
Код сам подхватит файлы, если они есть. Пока их нет — игрок рисуется красным квадратом (как в v0.1).

| Файл | Кадров в ряду | Назначение |
|------|----------------|------------|
| `player_idle.png` | 4 | стойка на месте |
| `player_walk.png` | 6 | ходьба |
| `player_hurt.png` | 2 | получение урона |

Пример размера листа: `player_walk.png` из 6 кадров по 64×64 → ширина **384**, высота **64**.

Отражение влево/вправо делает код автоматически — рисуй персонажа смотрящим **вправо**.
