# Ассеты (звук, музыка, шрифты)

Положи сюда свои сгенерированные файлы — игра подхватит их автоматически.
Если какого-то файла нет — просто не будет этого звука, игра не сломается.

## Шрифт (нужен для русского текста!)
- `assets/fonts/game.ttf` — любой TTF с кириллицей (например, PT Sans, Inter, Roboto, Fira Sans).
  Без него русский текст в меню и HUD покажется квадратиками.

## Музыка (assets/audio/)
- `music_menu` — музыка лагеря / главного меню
- `music_game` — музыка карты

Форматы: `.ogg`, `.mp3` или `.wav` (ищется именно в таком порядке).

## Звуки (assets/audio/)
- `shoot` — выстрел
- `hit` — игрок получил урон
- `pickup` — подбор опыта
- `levelup` — повышение уровня
- `boss` — появление босса

Форматы: `.wav`, `.ogg` или `.mp3`.

## Голоса боссов (assets/audio/)
- `voice_spider` — Королева пауков: «Ты будешь кормом моих детей.»
- `voice_knight` — Чёрный рыцарь: «Жалкая пародия.»

Система уже готова: при появлении босса показывается субтитр и играется голос (если файл добавлен).
